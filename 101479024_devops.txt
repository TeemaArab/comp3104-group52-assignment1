COMP 3104 - DevOps is a course at George Brown College that introduces students 
to the fundamental principles of DevOps. 
DevOps is a set of practices that combines software development (Dev) and IT operations (Ops) aimed at 
shortening the software development lifecycle and providing 
continuous delivery with high software quality.

#The second Commit

Key topics covered in COMP 3104 include:
- Continuous Integration (CI) and Continuous Deployment (CD) pipelines
- Version control with Git and GitHub
- Infrastructure as Code (IaC) using tools like Docker and Kubernetes
- Automation testing and monitoring tools
- Cloud services for deployment

#The Third Commit

The course emphasizes hands-on experience with tools and technologies that are widely used 
in the industry, preparing students for roles in modern software development environments.

#The Fourth Commit

Real-World Applications of DevOps:
DevOps practices are essential in modern software development, enhancing collaboration 
between development and operations teams. This course equips students with the skills to 
automate deployment, monitor application performance, and quickly recover from failures, 
preparing them for effective roles in the industry.
